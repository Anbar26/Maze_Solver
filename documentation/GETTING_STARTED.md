## Common Issues

### Agent hits walls?
- Increase episodes to 2000
- Make sure gamma is 0.99
- Try Q-Learning algorithm

### Training too slow?
- Reduce episodes to 500
- Use Q-Learning (fastest)
- Avoid Monte Carlo for quick results

### Want to create your own maze?
1. Click "Edit Mode"
2. Use tools to draw walls/paths
3. Click "Save" when done
4. Train on your custom maze!
